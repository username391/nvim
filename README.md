

#### Установка LSP
Для работы нужен установленный nodejs и пакетный менеджер npm

Установка `pyright`:
```shell
npm i -g pyright
```

Можно еще обновить что он там обновляет, просто на всякий
```vim
:MasonUpdate
```

#### Настройка темы
Комбинация клавиш `Space + t` открывает меню для выбора темы
