


### Установка LSP
Для работы нужен установленный nodejs и пакетный менеджер npm

Установка pyright:
```shell
npm i -g pyright
```

Можно еще обновить что он там обновляет, просто на всякий
```shell
:MasonUpdate
```


### Nvim-Tree
не забудь поставить символ для untracked в plugins/nvim-tree.lua
